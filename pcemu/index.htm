一:基于Wine

 

1.原生Wine(不推荐)

Wine（即 WineHQ）是一个 Windows 兼容层，允许你在类 Unix 操作系统（如 Linux）上运行 Windows 应用程序.

在安卓上,Wine也发布了版本,可在https://dl.winehq.org/wine-builds/android/查看有关消息

原生Wine需要使用外接键盘且对高版本安卓兼容性不是很好,不是很推荐使用

你知道吗: Wine = Wine Is Not an Emulator

 

2.Exagear(小游戏首选)

Exagear是一个基于Wine的Windows模拟器,由一个俄罗斯公司开发(似乎倒闭了),属于很多人(包括我)在安卓机上模拟Windows的启蒙软件,可能是使用旧版本Wine的原因,Exagear对于老游戏(比如红警2,帝国时代1)的兼容性以及性能似乎比Winlator或者Box86/64要好......

Exagear有好几个版本,比如ED,ET,EX版,平时一般使用ED版

只需要将游戏放入手机存储的Exagear目录下,进入Exagear ED,创建环境,启动环境,找到游戏打开即可游玩

ED302(直装版)下载地址:https://www.downkuai.com/android/163378.html

(玩红警2可以用Exagear Mobile(暂时不清楚是什么版本,好像是ES?))

 

3.Winlator(无特殊情况推荐)

希望在安卓设备上使用Windows软件或游戏?

请使用winlator软件 点击跳转详情页

Winlator是一个 Android 应用程序,可让您使用Wine和Box86/Box64

运行 Windows(x86_64)应用程序。

相比于Exagear,Winlator现在任然有人在做维护及升级Wine版本,因此可以运行的软件也就更多

Github页面 (国内可能无法打开)

 

4.Box86/Box64/Mobox

基本是目前性能最好的基于Wine的Windows模拟器,使用最新的Wine版本以及驱动

单独安装比较麻烦(甚至可能需要科学上网),建议去超玩游戏盒下载整合版,可以简化90%的操作

Github页面 (国内可能无法打开)

 

二:基于QEMU

QEMU是一种通用的开源计算机仿真器和虚拟器。QEMU共有两种操作模式

1.全系统仿真：能够在任意支持的架构上为任何机器运行一个完整的操作系统

2.用户模式仿真：能够在任意支持的架构上为另一个Linux/BSD运行程序

同时当进行虚拟化时，QEMU也可以以接近本机的性能运行KVM或者Xen。

 

简单来说就是提供一个系统镜像,在你的手机上安装一个完整的操作系统(或者提供磁盘文件直接启动系统)

 

在使用安卓上的QEMU时,一般我们使用原生QEMU或者Limbo,原生QEMU不是很兼容高版本安卓,且需要另外装VNC,所以这里只对Limbo进行讨论

 

1.下载系统镜像和limbo

Limbo在不使用虚拟化(KVM)的情况下性能很差,不建议在没有虚拟化的情况下运行要求较高的操作系统,以Microsoft Windows操作系统为例,最高不要超过Windows Vista,否则运行速度会非常慢,使用Windows Server系统将会有些许提升(个人认为Windows Server 2003及以下系统的体验较好)

推荐一个系统磁盘站:https://steve372a.github.io/images/

在下载前注意显卡、声卡等参数,在网站上会有标注

作者:百度贴吧:苹果论里小号

 

里面的Windows镜像还是很齐的(不知道为什么没有Windows ME)

Limbo6.0.0下载:https://www.ddooo.com/softdown/230108.htm

下载好自己需要的镜像后打开Limbo,把弹窗点掉后,点击右上角的None,点击New,输入虚拟机的名字(随意)

然后点最下面的Custom,然后点"用户界面"那一栏,把鼠标改成usb-tablet(鼠标),

打开"面板"那一栏,CPU型号一般选择core2duo,内存(RAM)根据自己的情况选择

下面的选项可以开启MTTCG,也可以试试开启KVM(虚拟化),若启动机器后立即闪退,那么可能是你的手机不支持虚拟化,请关闭KVM重试

磁盘那一栏选择下载到的磁盘文件

 

"图形选项"那里的"视频显示"就是显卡,选择网站上标注的显卡,若没有对应的显卡一般选择cirrus

"音频"里的声卡和"网络"里的网卡(需要网络模式使用user)同理,声卡的和网卡不是很重要,没有对应的直接选择"无"即可

 

配置好后就可以开机了,在开机后的界面内按手机的返回键调出菜单,可以使用手机的键盘以及调整一些参数.
